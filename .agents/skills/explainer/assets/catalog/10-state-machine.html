State-Machine Map

The ticket is a state machine. Boxes are the states the row moves through; the pills between them are transitions, each labelled with the condition that fires it (claim / classify / send / queue). Terminal states glow. Click any transition to see the code that drives it and the one line it writes. Flip LIVE ⇄ PREVIEW: every WRITE transition recolours inert in place — same computed path, nothing persisted.
